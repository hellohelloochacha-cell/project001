# project001

## 소개

이 프로젝트는 웹 애플리케이션 개발을 위한 코드 저장소입니다.
사용자 화면, 주요 기능, 데이터 처리 로직 등을 관리하며 서비스 개발에 필요한 코드를 포함합니다.

## 주요 기능

* 사용자 화면 구성
* 주요 서비스 기능 구현
* 데이터 처리 및 상태 관리
* 프로젝트 실행 및 배포를 위한 기본 구조 관리

## 사용 방법

1. 저장소 다운로드

```bash
git clone https://github.com/hellohelloochacha-cell/project001.git
```

2. 프로젝트 폴더로 이동

```bash
cd project001
```

3. 패키지 설치

```bash
npm install
```

4. 개발 서버 실행

```bash
npm run dev
```

5. 브라우저에서 접속

```bash
http://localhost:3000
```
